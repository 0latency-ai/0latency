## Recording Tips
- Use your mic, quiet room
- Phone or computer recording app (Voice Memos, Audacity, GarageBand — all work)
- Speak at a natural pace — slightly slower than conversation
- Pause 2 seconds between clips
- Don't worry about perfection — we can trim silence and adjust timing in edit
- Save as one continuous file OR separate files per clip (either works)

## File Naming
If separate: `clip-1-setup.m4a`, `clip-2-problem.m4a`, etc.
If one take: `voiceover-full.m4a`
