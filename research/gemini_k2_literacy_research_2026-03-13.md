## Key Findings (from initial output)

### Core Insight
K-2 literacy delivery = teacher-mediated instruction + physical manipulatives as PRIMARY vehicles. Digital tools are "classroom catalysts" that enhance, not replace, the human-centric core.

### The Fundamental Paradox
Digital platforms offer personalization and data-driven intervention, but K-2 students lack cognitive and motor autonomy to navigate them independently. Leading platforms solve this with hybrid ecosystems.

### Science of Reading Context
- Industry transitioning from "balanced literacy" to "structured literacy"
- Requires explicit, systematic, cumulative instruction
- K-2 = high-touch environments, teacher directs flow, students engage with physical artifacts (letter cards, chaining folders, decodable texts)

### Platform Analysis

**Amplify CKLA** — Dual-strand architecture:
- **Skills Strand:** Almost entirely physical manipulatives + teacher-directed routines. K students use Chaining Folders and Small Letter Cards (physically build/manipulate words). Grades 1-2 add Consonant/Vowel Code Posters and Individual Code Charts. Digital component = Sound Library (teacher projects articulation videos/songs)
- **Knowledge Strand:** Read-aloud as primary vehicle. Teacher uses Knowledge Flip Books and Image Cards for visual anchors. Students build vocabulary/content knowledge ORALLY before they can read independently.

## Implications for Project Explore Daytime Version

1. K-2 delivery should be teacher-led with animated episodes projected (similar to Knowledge Strand read-alouds)
2. Physical activity packets per episode (printable, differentiated by reading tier)
3. Teacher-only dashboard for assessment input (observation-based rubrics, not digital student assessments)
4. Digital student interaction begins at Grade 3
5. The 4 reading tiers map naturally to differentiated physical materials
6. Bilingual delivery = teacher toggles episode language, separate print packets per language pathway
