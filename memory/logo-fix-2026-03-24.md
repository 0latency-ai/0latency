# Logo Fix — 2026-03-24

## Problem
On mobile, the nav logo (`0latency-logo-light.jpg`) rendered too large and the full SVG version showed "0 Lat ency" with broken text due to `<tspan>` elements in the SVG.

## Solution
Replaced the nav logo across all 16 HTML pages with a responsive approach:

- **Desktop (≥768px):** `logo-icon.svg` (36px) + styled "0Latency" text (`<span class="logo-text">`)
- **Mobile (<768px):** Icon only — text hidden via CSS media query

### CSS added to each page:
```css
.logo-text { display: inline; }
@media (max-width: 767px) {
  .logo-text { display: none !important; }
  .logo img, .nav-logo img { height: 32px !important; }
}
```
