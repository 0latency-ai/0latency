## MAJOR FIX — Tier Value Delivery (6:00-6:03 PM UTC)

**Problem:** Free tier limit enforcement was broken. Exception handling bug was converting HTTP 429 errors to 500 errors.

**Root cause:** 
```python
except Exception as e:
    raise HTTPException(500, ...)
```
This was catching the `HTTPException(429)` raised by limit enforcement and re-raising it as 500.

**Fix:**
```python
except HTTPException:
    raise  # Re-raise HTTP exceptions as-is
except Exception as e:
    raise HTTPException(500, ...)
```

**Test proof:**
- Created account with 5-memory limit
- Added 5 memories successfully
- 6th attempt correctly returned HTTP 429 with message: "Memory limit reached (5). Upgrade plan or delete old memories."
