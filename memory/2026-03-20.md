### Memory Product Refinements Identified (Echo's proposals)

**Recall quality:**
- Recency weighting in recall — time-weighted ranking ("2 hours ago" beats "3 weeks ago")
- Recall feedback loop — reinforce memories that get used in responses, demote ones that get surfaced but ignored. System learns what's useful, not just what's relevant. **This is the highest priority refinement.**

**Extraction quality:**
- Intent detection — distinguish "stated a fact" vs "brainstorming" vs "venting." Different memory types and decay rates.
- Conversation phase awareness — early turns = context-setting, middle = working, final = decisions/summaries. Weight extraction accordingly.

**Operational:**
- Memory health dashboard — total count, extraction rate, recall hit rate, contradiction flags, orphaned memories
- Audit trail — "why did you think X?" traces to exact conversation turn. Source attribution in schema but needs exposure to agent.

**Scale prep:**
- Memory partitioning by domain (PFL, SSU, personal, infrastructure) — matters at 500+ memories
- Ephemeral memory type with aggressive TTL — "I'm at Waterbar tonight" useful for 12 hours, not 12 months. Forgetting with intent.
