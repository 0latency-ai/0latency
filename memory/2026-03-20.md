3. **Structured List Preservation** — Numbered lists preserved as single memories. 9-item checklist → 1 coherent memory, not 9 disconnected tasks. Tested: same input, 2 memories instead of 9.
4. **Deduplication Two-Tier** — 0.92+ = duplicate regardless. 0.85-0.92 = duplicate only if same type. Prevents cross-type false matches while catching near-dupes.
5. **Hybrid Recall (Semantic + Keyword)** — Strategy 4 added: keyword/text search catches what embedding similarity misses. Expanded stop word list.
6. **Correction Type Tightened** — Only "correction" when conversation explicitly says "X was wrong, it's actually Y." Reduces noise from 29% corrections to appropriate level.
