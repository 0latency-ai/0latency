Privacy Policy

Last updated: March 23, 2026

1. Introduction

0Latency ("we", "our", "us") operates the 0Latency memory layer API and related services at 0latency.ai. This Privacy Policy explains how we collect, use, store, and protect your information when you use our services.

2. Information We Collect

Account Information: When you create an account, we collect your name, email address, and authentication credentials (via GitHub, Google, or email/password).

Memory Data: When you use our API, we store the memory content you submit through the extract and add endpoints. This data is stored on your behalf and is accessible only to you and your authorized agents.

Usage Data: We collect API call counts, timestamps, and error rates for billing, rate limiting, and service improvement.

Technical Data: Standard server logs including IP addresses, user agents, and request metadata. These are retained for 30 days for security and debugging purposes.

3. How We Store Your Data

All data is stored in Supabase PostgreSQL databases hosted on AWS infrastructure. Memory data is encrypted at rest using AES-256 encryption. All API communication is encrypted in transit via TLS 1.3.

We do not store your passwords in plaintext. OAuth tokens are managed by the respective providers (GitHub, Google).

4. How We Use Your Data

  • To provide and operate the 0Latency memory layer service
  • To authenticate your API requests
  • To process billing and manage your subscription
  • To monitor and improve service performance and reliability
  • To communicate important service updates (outages, security issues, billing)

5. What We Don't Do

  • We never sell your data. Not to advertisers, data brokers, or anyone else.
  • We never use your memory content to train AI models. Your data is yours.
  • We never share your data with third parties except as required to operate the service (Supabase for storage, Stripe for billing).

6. Data Retention

Your memory data is retained as long as your account is active. If you cancel your subscription, your data remains accessible on the free tier. If you delete your account, all data is permanently deleted within 30 days.

7. Your Rights (GDPR-Friendly)

Regardless of where you are located, we provide the following rights:

  • Access: You can export all your data at any time via the API.
  • Correction: You can update or modify any memory via the API.
  • Deletion: You can delete individual memories or your entire account. Email privacy@0latency.ai for full account deletion.
  • Portability: All data is available via our REST API in standard JSON format.
  • Objection: Contact us to object to any specific data processing.

8. Third-Party Services

  • Supabase — Database hosting and authentication infrastructure
  • Stripe — Payment processing (we never see your full credit card number)
  • Google Gemini Flash 2.0 — Memory extraction processing (your data is processed but not retained by Google under their API terms)

9. Cookies

We use minimal cookies for authentication sessions only. We do not use tracking cookies, analytics cookies, or advertising cookies.

10. Security

We implement industry-standard security measures including encryption at rest and in transit, API key authentication with HMAC signing for webhooks, and regular security reviews. If you discover a security vulnerability, please report it to security@0latency.ai.

11. Changes to This Policy

We may update this policy from time to time. Significant changes will be communicated via email to registered users. The "Last updated" date at the top reflects the most recent revision.

12. Contact

For privacy-related inquiries, data deletion requests, or questions about this policy:

  • Email: privacy@0latency.ai
  • GitHub: Open an issue
