🧩 0Latency Chrome Extension

Automatically capture AI conversations from ChatGPT, Claude, Gemini, and Perplexity into your 0Latency memory layer.

⚠️ Pre-Launch Notice
The Chrome extension is currently in development. It will be available via Chrome Web Store after launch. For now, you can install it manually in developer mode.

📦 Installation (Developer Mode)

  1. Download the extension: Request access
  2. Open chrome://extensions in Chrome
  3. Enable Developer mode (top right toggle)
  4. Click Load unpacked
  5. Select the downloaded chrome-extension folder

⚙️ Setup

  1. Click the 0Latency icon in your Chrome toolbar
  2. Enter your API key (zl_live_...)
  3. Set your Agent ID (default: chrome-extension)
  4. Toggle auto-capture ON

✨ How It Works

The extension watches your AI conversations using DOM observers and detects conversation turns on supported platforms:

  • ChatGPT — Detects [data-message-author-role] elements
  • Claude — Detects [data-testid="user-message"] and [data-testid="ai-message"]
  • Gemini — Detects query/response containers
  • Perplexity — Detects question/answer pairs

When a conversation turn completes (2s debounce after streaming finishes), the extension sends the human message + agent response to your 0Latency API for memory extraction.
