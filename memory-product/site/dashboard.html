🚀 Quick Start

1 Install the MCP server:

npx @0latency/mcp-server@0.1.4

2 Add to your Claude Desktop config:

{ "mcpServers": { "0latency": { "command": "npx", "args": ["-y", "@0latency/mcp-server@0.1.4"], "env": { "ZERO_LATENCY_API_KEY": "your_key_here" } } } }

3 Restart Claude Desktop and start chatting!

Claude will now remember your preferences, decisions, and context across sessions.

📖 API Documentation

Use the REST API directly for custom integrations.

Store a memory:

curl -X POST https://api.0latency.ai/extract \ -H "X-API-Key: YOUR_KEY" \ -H "Content-Type: application/json" \ -d '{"agent_id":"my-agent","human_message":"I prefer React","agent_message":"Noted"}'

Recall memories:

curl "https://api.0latency.ai/memories?agent_id=my-agent&limit=10" \ -H "X-API-Key: YOUR_KEY"
